# Frogger Web Game

A simple web-based Frogger game built with HTML, CSS, and JavaScript.

## How to Play
- Use the arrow keys to move the frog up, down, left, and right.
- Avoid the cars and use the logs to cross the river.
- Reach the top row to score a point.
- The game ends if you collide with a car.

## How to Run
1. Open `index.html` in your web browser.
2. Play directly—no build or server required.

## Project Structure
- `index.html` — Main game page
- `scripts/game.js` — Game logic
- `styles/style.css` — Game styles
- `assets/` — Placeholder for images or sounds (optional)

## Notes
- All graphics are drawn with canvas and simple colors for clarity.
- You can enhance the game by adding images or sound effects in the `assets` folder.
